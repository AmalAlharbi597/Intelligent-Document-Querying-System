# Intelligent Document Querying System

This project allows users to intelligently query, extract, and summarize information from uploaded documents using AI models.It is designed to process various document types and provide relevant answers based on user queries.
NOTE Some of the PDF files in this project are too large to preview directly on GitHub.
If a file does not open in your browser (shows a loading circle), please download it manually using the “Download” button or the “View raw” option.
